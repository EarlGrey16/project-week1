
# Tech Repair Reservations Website

## Overview

A simple, one-page website for users to book appointments for phone or computer repairs.

## Current Features

*   **Reservation Form:** A form to collect user information, device type, and problem description.

## Design and Style

*   **Layout:** Centered content with a clear heading.
*   **Colors:** A simple color scheme with a primary color for the header and buttons.
*   **Typography:** Clean, readable fonts.
*   **Interactivity:** A button with a clear call-to-action.

## Plan for Current Request

1.  **Modify `index.html`:** Create the structure for the reservation form, including input fields for name, email, phone, device type, problem description, and a submission button.
2.  **Modify `style.css`:** Apply styles to the form, inputs, and button to ensure a clean and modern look.
3.  **Modify `main.js`:** Add an event listener to the form to handle submission, initially logging the data to the console.
